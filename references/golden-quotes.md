# 金句时刻（# 金句时刻）

挑选 1-4 句有代表性、有信息量或金句性质的原文原话：

```
「{原话，尽量保留原句措辞}」

—— {一句话说明这句的价值/含义}
```

## 选取规则

- 必须是文字稿里**真实出现**的话，不得改写意思，可略去口水词。
- **金句数量以 3 条为佳**（最多不超过 4 条），多数应精炼（≤40字），优先选一句话讲清道理/方法/共识的句子。**3 条最佳组合：定调句 / 方法论句 / 升华句 各一**。
- **优先选"点题句"**：与会议核心论点/最关键结论直接强相关的表述优先（如全篇题眼那句、全篇定调句），社会价值/感性金句最多 1 句作收尾，至少 2 句须点题。3 条金句宜覆盖：**定调句 / 方法论句 / 升华句** 各一。
- **优先收录被反复点题的句子**：若文字稿中某句话被讲者多次重复/强调（如"仰望星空脚踏实地""EGO 很高""正确的路只有一条"），应优先收录，避免用语义相近的其他句替换。
- **恰好 1 条可以是"核心落地思路/架构构想的较长原话"**（可至百余字）——仅当会议中确有一段被反复强调、构成核心结论的长发言时收录这 1 条。其余金句保持精炼，**不要把多条长论述都塞进来**。
- 兼顾不同说话人，宁缺毋滥。没有合适的就省略本章（连同其 readonly-block）。
